# PROJETO BANCO TERMINAL
## ADIÇÕES DO PROJETO

1. Sistema de escolha de criação de conta no inicio:
 
   Se o usuario escolher criar a conta o algoritmo inicia, se não apresenta uma mensagem e encerra o código

2. Sistema de criação de conta:
 
   Sistema padrão do projeto onde o usuário digita sua agencia, número de conta, seu nome e o quando deseja depositar inicialmente

3. Mensagem padrão de criação de conta:
 
   Após a criação da conta a mensagem de sucesso aparece, alem dela aparece o número da agencia, da conta, nome do usuário e seu saldo depositado
